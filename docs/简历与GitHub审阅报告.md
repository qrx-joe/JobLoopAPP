**上海**（T3，匹配度 2/5）

- 金融科技+SaaS公司多，经济学背景有加分
- 但985扎堆，竞争激烈
- 可用于投"金融科技公司"的产品/运营岗（工行实习有加分）
- 不建议投纯金融（券商/基金偏好硕士+CFA/CPA）

**广州**（T3，匹配度 2/5）

- 微信、网易、SHEIN，电商/社交产业发达
- AI产业密度在一线中最弱，产品岗偏传统互联网和电商运营
- 生活成本相对可控

**苏州**（T3，匹配度 2/5）

- 微软/苹果有分部，苏州AI产业园在发展
- 偏技术研发岗，产品/运营岗位少
- 可作为上海投递的"卫星城"

**合肥**（可关注，匹配度 2/5）

- 科大讯飞总部，AI产业有独特优势（语音AI方向）
- 中国科大的AI研究资源，但城市吸引力有限
- 如果对语音AI方向有兴趣，讯飞的产品岗值得投

**西安**（不推荐，匹配度 1/5）

- 华为/中兴有研发中心，但以硬件和通信为主
- AI产业薄弱，产品/运营岗位数量很少
- 虽然离山西最近，但岗位太少

**长沙**（不推荐，匹配度 1/5）

- 芒果TV、兴盛优选，互联网产业在成长
- AI产业几乎空白，生活成本极低
- 岗位方向不匹配

### 5.5 面试准备要点（产品/运营方向）

面试时你需要的核心故事线：

**故事1："我不是在学技术，我是在用技术解决真实问题"**

- JobMatch：考公选岗痛点 → 零上传工具 → 3天完成 → 已上线
- 这个故事证明你有产品嗅觉和快速落地能力

**故事2："GAP两年不是在空窗，是在做产品探索"**

- 9个项目 = 9次产品MVP验证
- 每个项目都始于真实场景的痛点观察
- 从"能用"到"好用"的迭代意识（公众号文章《同一个项目，我做了两遍》）

**故事3："经济学训练让我有数据思维"**

- 校创项目用DID/PSM-DID做因果推断

- 乡镇府实习做3万+人口数据治理

**故事4："内容运营实践让我理解用户增长"**
